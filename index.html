Ltecobike — Sistema de gestión para movilidad eléctrica

Sistema interno desarrollado para gestionar ventas, stock, clientes, gastos, distribuidores y catálogo web de una empresa de motos eléctricas. Combina backend PHP/MariaDB, panel administrativo, sitio público y despliegue sobre servidor Linux con Docker y Cloudflare.

v1 · 2025

Panel básico

PHP + MariaDB, Laragon. Validación del modelo de negocio.

v2 · 2025

Dockerización

Migración a Rocky Linux + Docker. Cloudflare Tunnel + Pages.

v3 · 2026

Funcionalidades principales

Roles, comisiones, QR, balance y Cloudflare Access.

Arquitectura de infraestructura

Panel y web pública desplegados en contenedores Docker sobre Rocky Linux. MariaDB instalada en el servidor host. Cloudflare Tunnel corre como servicio del sistema para exponer el panel sin abrir puertos. La web pública hace deploy automático desde GitHub.

FP

Soy de Montevideo, Uruguay. Empecé en informática en 2022 y desde entonces no paré. Lo que más me impulsa es resolver problemas reales con herramientas técnicas.

Ltecobike no empezó como portfolio. Empezó porque mi familia necesitaba un sistema para gestionar el negocio y decidí construirlo. Eso me forzó a tomar decisiones concretas: qué tecnología usar, cómo deployar de forma segura, cómo diseñar una base de datos que funcione en producción.

Me interesa la intersección entre desarrollo backend y administración de sistemas. Estoy al inicio de mi carrera y busco una oportunidad para seguir aprendiendo en un entorno profesional real.

Mis Proyectos

SIGTO, desarrollado bajo la compañía ficticia Sellxpress, es un avanzado Sistema de Gestión de Tiendas Online. Este proyecto fue diseñado para optimizar y gestionar las operaciones de comercios electrónicos mediante una plataforma robusta y segura.

El sistema fue construido utilizando C# como lenguaje de programación, junto con una base de datos SQL. Para garantizar un alto rendimiento y escalabilidad, la base de datos se implementó en un servidor Fedora y se ejecuta dentro de un contenedor Docker.

Servidor, Base de Datos y Docker

El sistema SIGTO está basado en un entorno de servidor robusto con Fedora como sistema operativo, y utiliza una base de datos en SQL, gestionada mediante MariaDB como gestor de bases de datos. Esta base de datos se encuentra alojada dentro de un contenedor Docker, lo que permite una gestión eficiente, portátil y escalable, facilitando su implementación en cualquier entorno compatible con Docker.

Gracias a Docker, el sistema puede escalar de manera optimizada, mientras que Fedora proporciona un entorno seguro y de alto rendimiento para ejecutar la aplicación. La configuración de la base de datos en un contenedor Docker permite replicaciones rápidas y restauraciones eficientes, garantizando la disponibilidad continua del servicio en todo momento.

Además, se integró un firewall externo basado en pfSense, configurado con reglas específicas para proteger los datos y garantizar la seguridad de las transacciones en línea.
